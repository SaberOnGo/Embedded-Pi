void xACMPConfigure ( unsigned long  ulBase,
unsigned long  ulCompID,
unsigned long  ulConfig 
)

Configures a comparator.

Parameters
ulBaseis the base address of the comparator module.
ulCompIDis the ID(index) of the comparator to configure. Refrence xACMP Analog Comparator ID.
ulConfigis the configuration of the comparator.

This function configures a comparator. The ulConfig parameter is the result of a logical OR operation between the ACMP_ASRCP_xxx, and ACMP_ASRCN_xxx values.

The ACMP_ASRCP_xxx is determines the ACMP+ source, values refrence xACMP Positive(ACMP+) Input Source.

The ACMP_ASRCP_xxx is determines the ACMP- source: values refrence xACMP Negative(ACMP-) Input Source.

Returns
None.
