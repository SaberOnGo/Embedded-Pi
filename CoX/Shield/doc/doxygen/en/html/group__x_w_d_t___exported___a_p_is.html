void xWDTInit ( unsigned long  ulBase,
unsigned long  ulConfig,
unsigned long  ulReload 
)

Configurate The WatchDog Timer(WDT)'s Timer Interval.

Parameters
ulBaseis the base address of the WatchDog Timer(WDT) module.
ulConfigis the Timer's Prescaler divider and clock source Selection. Details please refer to xWDT Clock Configuration and xWDT_Time_Reload_Config.
ulReloadis the Timer's reload value.

This function is to configureThe WatchDog Timer(WDT)'s Timer Interval. There are three factors to determine the Timer Interval, they are:

  • clock source
  • Prescaler divider
  • reload value

The ulConfig parameter contains clock source and Prescaler divider; When being used,this parameter is the logical OR of the two values. For NUC1xx,there is no Prescaler divider and the clock source can be: xWDT_S_INTSL, xWDT_S_EXTSL, xWDT_S_HCLK_DIV. Details please refer to 2.CoX Port Details.

The ulReload parameter is the value which will reload the counter when feeding the watch dog.The value range depends on the number of bits of corresponding counter. For NUC1xx, the reload value can only be one of the following: xWDT_INTERVAL_2_4T, xWDT_INTERVAL_2_6T, xWDT_INTERVAL_2_8T, xWDT_INTERVAL_2_10T, xWDT_INTERVAL_2_12T, xWDT_INTERVAL_2_14T, xWDT_INTERVAL_2_16T, xWDT_INTERVAL_2_18T. Details please refer to 2.CoX Port Details.

Note
When using watch dog xWDTEnable() should be called after call xWDTInit(ulBase, ulConfig, ulReload).
Returns
None.

void xWDTFunctionEnable ( unsigned long  ulBase,
unsigned long  ulFunction 
)

Enable the watch dog timer's function.

Parameters
ulBaseis the base address of the WatchDog Timer(WDT) module.
ulFunctionis the watch dog timer's function. Details please refer to xWDT Function Type.

This function is to enable the watch dog timer's function such as Interrupt reset up in ICE mode.

The ulFunction parameter is the OR value of any of the following: xWDT_INT_FUNCTION,xWDT_RESET_FUNCTION, xWDT_WAKEUP_FUNCTION,xWDT_HOLD_IN_ICE. Details please refer to
