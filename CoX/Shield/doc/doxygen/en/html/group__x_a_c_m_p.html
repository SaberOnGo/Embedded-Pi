Detailed Description

CoX ACMP Peripheral Interface.

The xACMP is the ACMP Peripheral Interface of CoX. It defines some common macros and APIs.

2. API Groups

The CoX ACMP API is broken into 2 groups of functions:

  • those that deal with the ACMP config,
  • those that deal with interrupts.

2.1 ACMP Config APIs

The ACMP is configured with:

  • xACMPConfigure() - configure the analog input source.

They are enabled and disabled with:

  • xACMPEnable()
  • xACMPDisable()

2.2 ADC Interrupt APIs

Event callback function init with:

  • xACMPIntCallbackInit()

Interrupt are enabled and disabled with:

  • xACMPIntEnable()
  • xACMPIntDisable()

3. Usage & Example

The following show how to use the ACMP APIs to initialize the ACMP.

  1. xSysCtlPeripheralEnable() to enable to GPIO, ACMP Peripheral clock.
  2. xSPinTypeACMP() to configure the Cn-, Cn+, Cno pin.
  3. xACMPConfigure() to configure the analog input source.
  4. xACMPEnable() to enable the ADC.
  5. xACMPIntEnable(), xIntEnable() to enable the interrupt.
