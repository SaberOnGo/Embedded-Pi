Detailed Description

CoX UART Peripheral Interface.

The xUART is the UART Peripheral Interface of CoX. It defines some common macros and APIs.

1. How is the xUART Standard Define?

CoX UART defines the APIs based on the following common functions:

  • Configure UART as Normal Mode
  • Configure UART to Support IrDA
  • UART to Support Modem Handshake
  • UART to Support LIN
  • Deal with the Interrupts


2. API Groups

The CoX UART API is broken into six groups of functions.


2.1 UART Normal Mode APIs

  • xUARTConfigSet()
  • xUARTEnable()
  • xUARTDisable()
  • xUARTFIFOEnable()
  • xUARTFIFODisable()
  • xUARTFIFORxLevelSet()
  • xUARTCharsAvail()
  • xUARTSpaceAvail()
  • xUARTCharGetNonBlocking()
  • xUARTCharGet()
  • xUARTCharPutNonBlocking()
  • xUARTCharPut()
  • xUARTBusy()
  • xUARTRxErrorGet()
  • xUARTRxErrorClear()


2.2 UART IrDA APIs

  • xUARTIrDAEnable()
  • xUARTIrDADisable()


2.3 UART Modem APIs

  • xUARTModemControlSet()
  • xUARTModemControlClear()
  • xUARTModemControlGet()
  • xUARTModemStatusGet()
  • xUARTFlowControlSet()
  • xUARTFlowControlGet()


2.4 UART LIN APIs

  • xUARTLINConfig()
  • xUARTLINEnable()
  • xUARTLINDisable()


2.5 UART Interrupt APIs

  • xUARTIntEnable()
  • xUARTIntDisable()
  • xUARTIntCallbackInit()
