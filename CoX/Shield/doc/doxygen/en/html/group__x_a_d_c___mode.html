Detailed Description

Values that show the ADC Operation Mode. 

They can be passed to 
xADCConfigure() as the ulMode parameter to configure the ADC.

Macro Definition Documentation

#define xADC_MODE_SCAN_SINGLE_CYCLE

Single cycle scan mode.

A/D conversion will sample and convert the specified steps once in the sequence, from the first step to the end. Then the A/D converter enters idle state.

#define xADC_MODE_SCAN_CONTINUOUS

Continuous scan mode.

A/D conversion will sample and convert the specified steps in the sequence, form the frist step to the end. and then The A/D converter repeat sample and convert from the first step, continuous.

when the mode is set as continuous scan mode, the ADC will begin sample after enable the ADC.
