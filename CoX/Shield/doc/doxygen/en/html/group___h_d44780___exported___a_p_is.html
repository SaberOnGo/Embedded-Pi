void HD44780Init ( void  )

Init the HD44780 LCD Device.

Parameters
None.This function is used to Init the HD44780 Device. It must be call before any other LCD function use.

It Open the pins's GPIO peripheral port, and config the pins type to GPIO output. Then config the LCD into the default state, clear the LCD and Open the display. Default A Blink cursor is set on. The LCD cursor move direction is config as increment default.

The HD44780 Initial state can be determined by the HD44780 Driver Configurtion.

  • Pins that used is determined by HD44780 Driver Pin Configurtion.
  • The inteface data length is determined by HD44780_INTERFACE_DATA_LEN.
  • The LCD display line is determined by HD44780_DISPLAY_LINE.
  • The LCD character font is determined by HD44780_CHARACTER_FONT.
Returns
None.

void HD44780EntryModeSet ( unsigned char  ucCursorMoveDirection,
xtBoolean  bDisplayShift 
)

Set the the LCD entry mode.

Parameters
ucCursorMoveDirectionis the cursor move dirction.
bDisplayShiftdetermines if the entire LCD shift when display a char.

In the HD44780Init() function, it has set a default entry mode. The cursor move direction is increment and no shift when display. You can change the setting with this function.

The parameter of ucCursorMoveDirection can be:

  • HD44780_MOVE_DIRECTION_INC - increment
  • HD44780_MOVE_DIRECTION_DEC - decrment.

The parameter of bDisplayShift cab be:

  • xtrue - display with shift.
  • xfalse - no shift wen display.
Returns
None.
