void xPWMInitConfigure ( unsigned long  ulBase,
unsigned long  ulChannel,
unsigned long  ulConfig 
)

Initialize and configure the PWM module.

Parameters
ulBaseis the base address of the PWM port.
ulChannelis the PWM channel.
ulConfigis the configuration of PWM channel. Details please refer to xPWM Config Type.

This function is to initialize and configure channel of the PWM module.

The ulChannel parameter can be values: 0~7,when the ulBase is PWMA_BASE,the ulChannel parameter can be value 0~3,when the ulBase is PWMB_BASE,the ulChannel parameter can be value 4~7.

The ulConfig parameter is the logical OR of four values: The PWM mode, Inverter or not,and use dead zero or not and dead zone length. xPWM_ONE_SHOT_MODE,xPWM_TOGGLE_MODE is the mode selecet. xPWM_OUTPUT_INVERTER_EN, xPWM_OUTPUT_INVERTER_DIS is to enable Inverter or not. xPWM_DEAD_ZONE_EN xPWM_DEAD_ZONE_DIS is to enable dead zone Generator or not. The dead zone length will be set at the 16~23 bits of the ulConfig. Details please refer to 2.CoX Mandatory and CoX Non-mandatory.

Note
When Dead-Zone Generator is enabled, the pair of PWM0 and PWM1 becomes a complementary pair for PWM group A and the pair of PWM4 and PWM5 becomes a complementary pair for PWM group B.
Returns
None.

Referenced by MSDCMotorPWMInit().
