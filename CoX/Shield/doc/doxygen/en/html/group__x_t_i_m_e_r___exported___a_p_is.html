void xTimerInitConfig ( unsigned long  ulBase,
unsigned long  ulChannel,
unsigned long  ulConfig,
unsigned long  ulTickFreq 
)

Configurate The Timer's mode and tick frequency.

Parameters
ulBaseis the base address of the Timer port.
ulChannelis the channel of the Timer port. Details please refer to xTIMER Count Channel
ulConfigis the mode Configuratation of the Timer port. Details please refer to xTIMER Count Direction and xTIMER Mode Type
ulTickFreqis the tick frequency of the Timer port.

This function is to configurate The Timer's mode and tick frequency.

The ulConfig parameter is the OR value of two values:

  • Timer's mode Details please refer to 2. CoX Port Details
  • Timer's direction Details please refer to 2. CoX Port Details

xTIMER_MODE_ONESHOT, xTIMER_MODE_PERIODIC, xTIMER_MODE_TOGGLE and xTIMER_MODE_CONTINUOUS is the Timer's mode

xTIMER_COUNT_UP, xTIMER_COUNT_DOWN is the Timer's direction.

The ulTickFreq parameter is the the timer tick frequency.It can be any values but ulTickFreq > 0 and ulTickFreq < timer clock source.

Note
For nuc1xx,each timer only one channel.
Returns
None.

Referenced by MSDelay().
